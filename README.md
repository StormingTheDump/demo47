# 匿名留言板 Demo

一个本地运行的 Vite + React + TypeScript 前端 Demo，用于内部员工匿名留言展示。

## 功能

- 4 个栏目：工作类、生活类、活动类、CEO 信箱
- 留言卡片展示分类、花名、发布时间、标题、正文、点赞数、评论数
- 分类 Tab、标题/正文搜索、最新/最热/评论最多排序
- 留言列表为单列信息流布局，卡片展示头像、花名、智能时间和可点击分类气泡
- 发布框仅允许匿名发布，支持分类选择、emoji 插入、图片选择、视频选择和话题插入
- 图片单次最多选择 9 张，视频单次最多选择 1 个，本地 Demo 仅保存附件摘要
- 同一浏览器内每条留言只能点赞一次，再点取消
- 评论按钮可展开/收起评论区，评论即时展示
- 有评论的留言会自动展示评论区
- 评论支持悬浮选择多种表情点赞，并允许继续回复评论
- 当前浏览器中新发布的测试评论可通过右下角更多菜单删除
- 更多菜单支持收藏、举报、复制链接的 demo 反馈
- CEO 信箱带独立视觉标识和匿名提交提示
- 橙色系内部工具视觉风格

## 运行

```bash
npm install
npm run dev
```

浏览器打开终端中显示的本地地址，通常是 `http://localhost:5173/`。

## 验证

```bash
npm run lint
npm run test
npm run build
```

## 数据保存

项目没有真实后端。初始内容来自 `src/data/mockMessages.ts`，用户新增的留言、评论和点赞状态会保存到浏览器 `localStorage`。

图片和视频是本地 demo 占位功能，只记录选择数量或文件名摘要，不会真正上传文件。

使用的 key：

- `anonymous-board:messages`
- `anonymous-board:liked-message-ids`
